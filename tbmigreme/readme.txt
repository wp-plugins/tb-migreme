O TB Migre.me encurta as suas urls com o Migre.me e publica na sua conta do twitter. O plugin ainda pode inserir o botão do Twitter em seus posts.

== Description ==

Com o TB Migre.me você consegue publicar seus posts no Twitter direto da tela de edição de posts do WordPress, utilizando o Migre.me como encurtador URLs. Ele ainda pode inserir automaticamente o botão oficial do twitter nos seus posts.

Outras funções:

* tb_migreme(); - Para inserir o botão de twitter manualmente (exibirá o botão escolhido no painel).

* get_tb_migreme(); - Se você precisar apenas da url do post encurtada pelo Migre.me, atribua essa função a uma variável.

* tb_migreme_form(); - Essa função imprime um form com a url do Migre.me dentro.

Veja a aba "Screenshots" para mais informações sobre o funcionamento do TB Migre.me

Obs: Esse plugin foi desenvolvido pelo **Tecnoblog**, não sendo oficial do **Migre.me**.

== Installation ==

1. Suba a pasta `tbmigreme` no diretório `/wp-content/plugins/`
1. Ative o plugin na aba de 'Plugins' do WordPress
1. Configure o plugin na aba 'Settings/TB Migre.me'

== Screenshots ==

1. Coloque aqui o seu login e senha do Twitter;
2. Selecione o botão que você quer que seja inserido no post. O plugin pode inserir automaticamente, ou você pode inserir manualmente utilizando a função tb_migreme();
3. Essa caixa aparece na tela de edição do post. Escreva algo para customizar o tweet ou deixe em branco para utilizar o título do post;
4. Colocando a função tb_migreme_form() no seu template, é isso que ela vai exibir.
